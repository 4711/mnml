A light-weight template for reponsive, HTML5 projects.

Fork on github Download project as .zip

MNML is readable by default.

MNML includes:

  • A bare-bones HTML5 file
  • Some light-weight sass organization
  • Normalize.css
  • Media queries for mobile & tablet devices
