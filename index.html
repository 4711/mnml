mnml

A light-weight template for responsive, html5 / scss projects.

Fork on github
Download as .zip

What is it?

Features

HTML5 template

Just the basics. An HTML5 doctype. A single css file include. A title tag and some meta tags. It's waiting for your content.

Basic scss setup

Normalize.css

Normalize.css is an alternative to reset.css. It preserves useful browser defaults - and only "targets the styles that need normalizing." See the project website for more info.

Media queries for mobile & tablet devices

Media queries for two breakpoints. Widths are easily configurable. Queries can be extended if needed.

Rake tasks

Simple rake tasks for starting sass for development as well as a css minification task to use before deployment to production.

Start prototyping faster. Don't spend time getting up and running. Use MNML.
