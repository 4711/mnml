MNML

MNML is a jumping off point for new responsive HTML5 projects. It's not a framework.

It includes:

  • A bare-bones HTML5 file
  • An HTML5 element only reset.css
  • A collection of media queries for various breakpoints and orientations.

Great to use as a sandbox for new responsive patterns as well as any other html/css development.
